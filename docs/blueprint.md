## Core Features:

- Raffle Number Display: Display 500 raffle numbers (001-500) in a grid format, differentiating between available and sold numbers. Numbers already sold should be displayed in gold and available in gray/black.
- Real-time Counter: Show in real time how many raffle tickets are left using a counter. Display the corresponding percentage sold using a progress bar.
- WhatsApp Purchase Button: Include a prominent green WhatsApp button that opens a chat with a pre-filled message for purchasing raffle tickets. This includes the desired WhatsApp phone number to start a conversation with.
- Pricing Table: Display the pricing options for buying raffle tickets: R$10 each, 3 for R$25, 7 for R$50.
- Rules and Transparency Section: Include a section detailing the raffle rules and ensuring transparency. Provide information such as live drawing details, proof of raffle process, etc.
- Responsive Design: The site will adjust to different device sizes with the primary focus on vertical format on mobile, with fast loading speeds and smooth experience across all devices.

## Style Guidelines:

- Primary color: Neon gold (#FFD700) for details and highlights.
- Background color: Dark desaturated green (#1A330A) for a dark theme inspired by the Brazilian soccer channel's aesthetic. Apply a darker green gradient to the black background for depth.
- Accent color: Lime green (#32CD32) for the 'purchase' button and other CTAs to draw attention.
- Headline font: 'Bebas Neue' (sans-serif) for titles. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' (sans-serif) for text, using a bold weight. Note: currently only Google Fonts are supported.
- Employ emojis relevant to the theme, like a sloth (mascot), soccer ball, fireworks, and money notes to enhance visual appeal. Each listed requirement will also get a small icon.
- The layout will be mobile-first, designed for a vertical screen (1080x1920). The mascot appears either at the top or bottom.
- Subtle animations will be applied when interacting with the purchase options and counter.
